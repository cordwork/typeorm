## Description

[TypeORM](https://typeorm.io/) module for [CordWork](https://github.com/cordwork/core).


## Getting Started

### Install
```
$ npm install @cordwork/typeorm
```

### tsconfig.json
```json
...
"experimentalDecorators": true,
"emitDecoratorMetadata": true,
...
```

* To check out the [example](/examples/).
